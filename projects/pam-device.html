Overview

For my individual dissertation in the third year of my degree, I developed a platform that was capable of connecting to a variety of sensors via I2C with a custom data package structure. This data could then be stored on an SD card and viewed by the user on an accompanying progressive web application. For this project, I developed a custom PCB with a microcontroller, accelerometer, real time clock, SD card socket, and power management system on board. This was all enclosed in a custom 3D printed case, with an external heart rate monitor developed to prove the capabilities of the system. The web application was developed from scratch using the web bluetooth standard to communicate with the main device. The full report of this project can be found below.
